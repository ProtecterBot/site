# 指令

# .purge

  • # 參數: .purge <數量: 數字>

  • # 作用: 按照最新訊息刪除訊息

# .bl

  • # 參數: .bl <選擇: +|add 或 -|del> <用戶>

  • # 作用: 將用戶 加入或移除 黑名單

選項 效果
+ | add 將用戶加入黑名單
- | del 將用戶從黑名單中移除

TIP

<用戶> 可以是提及或者用戶id
黑名單內的用戶加入伺服器後 會被再次移除

# .p

  • # 作用: 開啟或關閉 頻道保護

TIP

預設關閉 第一次輸入後會開啟 再次輸入會關閉 以此類推
防止頻道被刪除 刪除後會立刻重新創造權限設定一樣但歷史紀錄不一樣的頻道

# .igg

  • # 作用: 開啟或關閉 忽略頻道
